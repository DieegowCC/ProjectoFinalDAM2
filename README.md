
# Sett

Aplicación destinada al control de las acciones del trabajador/estudiante. Los trabajadores tendrán la aplicación de escritorio funcionando de fondo (un servicio), y va a recoger datos sobre su actividad y las aplicaciones que tiene de fondo abiertas además de a lo mejor los tiempos de descanso. Todo esto se va a mostrar por pantalla en la interfaz web (tipo dashboard) de administrador por clasificación. 
## Tech Stack

**Admin Dashboard:** React

**Client:** Windows Service, SignalR Client, Zustand

**Server:** Ubuntu Server, PostgreSQL, ASP.NET Core
